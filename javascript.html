O que é JavaScript?

JavaScript é a linguagem de programação mais popular do mundo, juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias padrões da web. E fácil de aprender

Por que estudar JavaScript?

JavaScript é uma das três linguagens que todos os desenvolvedores da web devem aprender:

  • HTML para definir o conteúdo das páginas da web.
  • CSS para especificar o layout das páginas da web.
  • JavaScript para programar o comportamento das páginas da web.

O que podemos fazer com JavaScript

  • JavaScript pode alterar valores de atributos HTML
  • JavaScript pode alterar estilos HTML (CSS)
  • JavaScript pode ocultar elementos HTML
